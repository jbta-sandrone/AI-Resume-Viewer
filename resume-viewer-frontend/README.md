# AI Resume Viewer - Frontend (React)

## Setup
1. Install Node.js 18+
2. Create and run:
   ```bat
   cd resume-viewer-frontend
   npm install
   npm run dev
   ```

Frontend runs by default on `http://localhost:5173`.

## Configure Backend URL
Edit `src/api/client.ts` and set `VITE_API_BASE_URL` if needed.
